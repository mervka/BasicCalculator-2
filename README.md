Basic Calculator Version-2

This project expands the basic class example with several new features, improved UI, and complete operation logic.

Features

Basic Operations
Addition (+)
Subtraction (–)
Multiplication (×)
Division (÷)

Extended Operations
Square (x²)
Square Root (√)
Decimal point ( . )
Decimal number support

Memory / Clearing Functions
C — Clear All
CE — Clear Entry

Smart Expression Preview
Displays the full operation above the result (e.g. 8 + 5, x²(5) + 3, etc.)

Two-number input
Users enter first and second numbers, select operations, and press "=" to calculate.
