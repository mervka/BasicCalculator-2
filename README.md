# Basic Calculator — Version 2

This project extends the basic calculator we developed in class and adds new functions, UI improvements, and a cleaner overall structure.

## Features

### Basic Operations
- Addition (+)  
- Subtraction (−)  
- Multiplication (×)  
- Division (÷)

### Extended Operations
- Square (x²)  
- Square Root (√)  
- Decimal numbers  
- Decimal point button (.)  

### Clearing Functions
- **C** → Clear All  
- **CE** → Clear Entry  

### Smart Expression Display
The calculator shows the full expression above the result.  
For example:  
- `8 + 5`  
- `5 × 5²`  
- `√25 + 3`  

### Two-Number Calculation Flow
Users enter:
1. First number  
2. Select an operation  
3. Second number  
4. Press “=” to calculate the result  

## How to Run the Project

1. Clone or download the repository.
2. Open **GuessGameApp.sln** with Visual Studio or JetBrains Rider.
3. Ensure that **.NET 8 SDK** and **MAUI workloads** are installed.
4. Run the project using an Android emulator or a connected device.
